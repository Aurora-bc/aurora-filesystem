# Aurora File System

一个基于 Go 语言开发的轻量级文件管理系统，支持用户注册登录、文件上传下载、文件元信息管理、文件查询、文件删除、秒传及下载链接生成等功能。项目采用 HTTP 路由方式组织接口，适合作为 Go 后端开发练习项目、课程设计项目或文件服务系统的基础示例。

## 项目简介

Aurora File System 面向基础文件服务场景，提供从用户认证到文件管理的一整套后端接口能力。系统围绕文件生命周期展开设计，覆盖文件上传、查询、更新、下载、删除等常见操作，同时预留了分块上传能力，便于后续扩展大文件传输与断点续传相关功能。

## 功能特性

- 用户注册与登录
- 用户信息查询
- 文件上传
- 文件上传成功跳转
- 文件元信息查询
- 文件列表检索
- 文件下载
- 文件元信息更新
- 文件删除
- 文件秒传
- 文件下载链接生成
- 预留分块上传扩展能力
- 静态资源访问支持

## 技术栈

- **语言**：Go
- **协议**：HTTP
- **项目结构**：基于标准库 `net/http` 路由分发
- **模块管理**：Go Modules

## 项目结构

```text
.
├── main.go          # 项目入口，负责路由注册与服务启动
├── go.mod           # Go 模块定义
├── go.sum           # 依赖校验文件
├── handler/         # 业务处理逻辑（如用户、文件相关接口）
├── static/          # 静态资源目录
└── README.md        # 项目说明文档
```

## 接口概览

### 用户相关

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/user/signup` | 用户注册 |
| POST | `/user/signin` | 用户登录 |
| GET | `/user/info` | 查询用户信息 |

### 文件相关

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/file/upload/` | 文件上传 |
| GET | `/file/upload/success` | 上传成功页 |
| GET | `/file/meta` | 获取文件元信息 |
| GET | `/file/query` | 查询文件 |
| GET | `/file/download` | 文件下载 |
| POST | `/file/update` | 更新文件元信息 |
| POST | `/file/delete` | 删除文件 |
| POST | `/file/fastupload` | 文件秒传 |
| GET | `/file/downloadurl` | 获取下载链接 |

### 静态资源

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/static/` | 访问静态资源目录 |

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/Aurora-bc/aurora-filesystem.git
cd aurora-filesystem
```

### 2. 安装依赖

```bash
go mod tidy
```

### 3. 运行项目

```bash
go run main.go
```

或编译后运行：

```bash
go build -o aurora-filesystem
./aurora-filesystem
```
